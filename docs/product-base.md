# Base de producto (contrato MVP)

Este documento fija las decisiones del proyecto. Cambios fuera de este alcance requieren justificación explícita.

## Funcionalidades incluidas

1. CRUD de recetas (crear, editar, eliminar, reutilizar)
2. Planificación semanal o quincenal con slots:
   - breakfast
   - lunch
   - dinner
3. Generación automática de shopping list:
   - sumar ingredientes repetidos
   - normalizar unidades
   - convertir a unidades de compra
4. Shopping list editable:
   - borrar ítems
   - ajustar cantidades
   - agregar ítems libres

## Funcionalidades fuera de MVP

- Recetas públicas/compartidas
- Pantry inteligente
- Recomendaciones automáticas
- IA/ML
- Información nutricional
- Importación externa de recetas

## Principio de UX

El usuario no gestiona conversiones manualmente. El sistema guía unidades y cantidades válidas.
